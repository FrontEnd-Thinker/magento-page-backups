SonicWall’s SonicWave 681 access point utilize 802.11ax technology, which provides for improved performance in high-density environments. SonicWave APs enhance the user experience in a number of ways. Not only are processor speeds faster, but beamforming allows for a more direct connection that is faster and more reliable than without beamforming. Additionally, 802.11ax works in both the 2.4 GHz and 5 GHz bands, and latency is reduced by 75%
