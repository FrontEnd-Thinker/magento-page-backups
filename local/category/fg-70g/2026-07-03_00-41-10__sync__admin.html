FG-70G / 71G FAQ

The hardware, port layout, and performance specs are identical. Both use Fortinet's SP5 ASIC, carry 10 x GE RJ45 ports, and run the same FortiOS feature set. The 71G adds a 64 GB onboard SSD, which gives the appliance local storage for logs, reports, and traffic history. If you want on-box visibility without sending everything to a FortiAnalyzer, the 71G is the right pick. If local storage is not a priority, the 70G does the same security job at a lower hardware cost.

Fortinet rates the 70G and 71G for 25 to 100 concurrent users. That covers a branch office, a retail location, or a small campus with room to grow. Sites with more than 100 users, or with heavy SSL inspection or threat protection loads, should consider the 90G or 120G instead. Our engineers can help you size the right model based on your actual user count and traffic profile.

Yes. SD-WAN is built into FortiOS and ships with every FortiGate appliance at no extra cost. The 70G and 71G use the SP5 ASIC to handle SD-WAN processing in hardware, so link balancing, application steering, and WAN failover run without taxing the CPU. ZTNA enforcement and a wireless controller for FortiAP management are also included in FortiOS, with no additional license required for either feature.

Fortinet offers three main subscription tiers for the 70G family. Enterprise Protection is the most complete, covering IPS, advanced malware protection, application control, URL and DNS filtering, DLP, anti-spam, FortiConverter, and FortiCare Premium support. Unified Threat Protection (UTP) includes IPS, antivirus, web filtering, application control, and FortiCare Premium. Advanced Threat Protection (ATP) focuses on IPS, advanced malware, application control, and FortiCare Premium without the URL and DNS filtering layers. Each bundle is available in one-year, three-year, and five-year terms.

Yes. FortiManager gives you a single console for policy management, firmware updates, reporting, and automated configuration across all your FortiGate appliances, including the 70G and 71G. FortiGate Cloud is the hosted alternative if you prefer not to run FortiManager on premises. Both options support zero-touch provisioning for fast branch deployment, and both integrate with the rest of the Fortinet Security Fabric for shared threat intelligence and unified visibility.
